よくある質問
気になるポイント
❓ 本当に無料ですか？
完全無料です。月額も従量課金もありません。NOU 自体が MIT ライセンスのオープンソース、内部で使う AI モデルも無料公開されているもの（Qwen, Llama, Gemma 等）を使います。必要なのはあなたの Mac だけです。
🔒 本当に会話は外に送られませんか？
送られません。AI モデルはあなたの Mac の中にダウンロードされ、チャットは全てローカルで処理されます。NOU はオープンソースなので、コードを誰でも検証できます。機内モードでも動きます。
🆚 ChatGPT や Claude と比べてどう？
NOU の強み: 無料・無制限・プライベート・オフライン対応・API料金ゼロ。
ChatGPT/Claude の強み: 最新最強モデル (GPT-4/Claude Sonnet) の生の性能。

日常のチャット・コーディング・要約ならNOUで十分です。最先端モデルが絶対必要な時だけ ChatGPT を使う、という併用もおすすめ。
💻 どんな Mac で動きますか？
Apple Silicon (M1 以降) のMacで動きます。メモリ別の目安:
・8GB: 軽量モデル (Qwen3.5 4B など) で快適
・16GB: 中型モデル (Qwen3.5 14B) で日常使用に十分
・32GB+: 大型モデル (Devstral, Qwen3-Coder) で GPT-4 に迫る品質
Intel Mac と Linux/Windows 対応は現在ベータです。
💾 どれくらい容量を使う？
NOU 本体は約 30MB。AI モデルは選ぶものによって 3〜50GB 程度。迷ったら 16GB Mac なら qwen3.5:14b (9GB) が安定しておすすめです。必要なら後から追加・削除できます。
🔥 Mac は熱くなりますか？
AI が返答を生成する数秒間だけ CPU/GPU を使います。生成が終われば通常モードに戻るので、長時間熱くなることはありません。Apple Silicon の Neural Engine を使うので電池の消費も比較的少なめです。
📱 iPhone からも使えますか？
はい。同じ Wi-Fi 内の Mac で NOU を動かしていれば、iPhone から自動で接続できます。外出先からもアクセスできるよう設定することも可能です。TestFlight で配布中です。
🛠 Claude Code / Cursor で使えますか？
はい、そのまま使えます。NOU は OpenAI 互換 API と Anthropic 互換 API の両方を提供しています。Claude Code なら環境変数に NOU のURLを設定するだけで、API 料金ゼロで使えるようになります。設定はインストール時に自動化されます。
