① インストール

1

GitHubから最新版をダウンロード

GitHub Releases から NOU-vX.X.X.dmg をダウンロードします。

2

DMGを開いてApplicationsにドラッグ

DMGを開き、NOU.app を Applications フォルダにドラッグします。

3

初回起動時の確認

Gatekeeper の確認ダイアログが出たら「開く」を選択します。
システム設定 → プライバシーとセキュリティ から「このまま開く」でも対応できます。

4

メニューバーのNOUアイコンを確認

起動するとメニューバーに「⬡ NOU」が表示されます。これで準備完了です。

💡 対応環境

macOS 14 Sonoma 以降、Apple Silicon (M1〜M5) が必要です。Intel Mac は非対応です。M5チップ（2025年発表）も完全動作確認済みです。

② モデルのダウンロードと起動

メニューバーの「⬡ NOU」をクリックして「ダッシュボードを開く」を選択します。

1

「モデル」タブを開く

ダッシュボードの「モデル」タブを開くと、お使いのMacのRAMに合わせたおすすめモデルが表示されます。

2

モデルをダウンロード

「⬇ ダウンロード」ボタンを押します。Qwen3-7B (〜5GB) の場合は数分で完了します。

3

「▶ 起動する」ボタンを押す

ダウンロード完了後、「▶ 起動する」を押します。数秒〜30秒でモデルがロードされます。

③ Claude Code / Cursor / Aider に接続

NOUはOpenAI互換APIとしてポート 4001 でリッスンしています。

🤖
Claude Code
ANTHROPIC_BASE_URL
⚡️
Cursor
Override Base URL
🛠️
Aider
--openai-api-base

Claude Code の設定

# ~/.bashrc または ~/.zshrc に追加 export ANTHROPIC_BASE_URL=http://localhost:4001 export ANTHROPIC_API_KEY=local-key-not-needed

設定後、新しいターミナルを開いて claude コマンドを実行します。

Cursor の設定

# Cursor Settings → AI → Override Base URL http://localhost:4001/v1

Aider の設定

aider --openai-api-base http://localhost:4001/v1 \ --openai-api-key local \ --model openai/auto

その他のOpenAI互換ツール

# 汎用設定 export OPENAI_API_BASE=http://localhost:4001/v1 export OPENAI_API_KEY=local

💡 モデル名について

モデル名に auto を指定すると、NOUが最適なモデルを自動選択します。特定のモデルを指定したい場合は qwen3-7b のように指定します。

④ iPhoneから使う (NOUConnect)

NOUConnect は、LAN内のNOUノードにiPhoneからアクセスできるクライアントアプリです。

1

App Storeからインストール

TestFlightから NOUConnect をインストールします。（近日リリース予定）

2

自動検出を許可

「ローカルネットワークへのアクセスを許可しますか？」が出たら「許可」を選択します。

3

ノードを選んでチャット開始

同じWi-Fiに接続されたNOUノードが自動で表示されます。タップして接続すると、iPhoneからAIが使えます。

⑤ 複数Macで分散推論

詳しくは 分散推論のページ を参照してください。簡単な手順は以下の通りです。

  1. すべてのMacにNOUをインストール
  2. 同じLANに接続
  3. ダッシュボードの「ネットワーク」タブから他ノードを確認（Bonjour自動発見）
  4. 「🖥️ ワーカーに追加」ボタンを押す

⑥ DePIN報酬を有効にする

使っていない時間にMacをDePINネットワークに開放することでNCH（NOU Compute Hours）報酬が積み立てられます。 設定はダッシュボードから1クリックで完了します。

1

「DePIN・報酬」タブを開く

ダッシュボードの「DePIN・報酬」タブを開きます。

2

「リレーに接続」ボタンを押す

ボタンを1クリックするだけで、NOUリレー（nou-relay.fly.dev）経由でノードが公開されます。URLは再起動しても変わりません。

3

報酬が自動で積み立てられる

他のユーザーのAI処理が来ると、処理したトークン数に応じてNCHが自動計上されます。

🔌 NOUリレーについて

CloudflareのTunnelは使いません。NOUが独自に運営するWebSocketリレー（Fly.io 東京リージョン）を使用します。ペアリングトークン認証とランダムノードIDでアクセスを制御します。

FAQ

Q: Intel Macでは動きませんか？

A: 現在はApple Silicon (M1以降) のみ対応しています。Intel Macはサポートしていません。

Q: 常時起動させておいても問題ない？

A: モデルが起動中でない限り、NOUのメモリ使用量は非常に少ないです。メニューバーアプリとして常駐させておくことができます。

Q: モデルはどこに保存される？

A: ~/Library/Application Support/NOU/models/ に保存されます。不要になったらここから削除できます。

Q: macOS 13 Ventura では動きますか？

A: macOS 14 Sonoma 以降が必要です。macOS 13では動作しません。

Q: セキュリティは大丈夫？

A: デフォルトではlocalhost (127.0.0.1) にのみバインドします。LAN経由のアクセスは認証が必要です。すべての処理はMac内で完結し、クラウドには送信しません。
